## Part 0: Philosophical & Historical Terms

### CCRU (Cybernetic Culture Research Unit)

Experimental cultural theorist collective (1995-2003) at Warwick University, England. Led by Nick Land and Sadie Plant, developed "theory-fiction" blending philosophy, cybernetics, and occult numerology.

**Key Contributions**: Numogram, Hyperstition, Pandemonium System
**Relevance**: Possible philosophical foundation for Qubic/Aigarth
**See**: [Background - Era 0](/docs/01-introduction/02-background), [Numogram Connection](/docs/03-results/29-numogram-architecture)

### Helix Gates

Ternary logic gates that are reversible and functionally complete, rotating inputs by sum of three ternary values.

**Origin**: Nick Land's Numogram (CCRU, 1990s)
**Implementation**: Core operation in Qubic/Aigarth
**Significance**: IDENTICAL terminology in both systems (probability < 0.1% if independent)

### Hyperstition

CCRU concept: "experimental techno-science of self-fulfilling prophecies." Ideas that make themselves real through belief and action.

**Examples**: Time-locks, breadcrumb trails, Numogram → Aigarth implementation

### Numogram

"Decimal Labyrinth" system (CCRU, 1990s): 10 digits organized into 5 syzygies, operating through feedback loops and time-circuits.

**Structure**: 10 zones, 5 syzygies, 3 time-systems, Helix gates, Pandemonium System
**Qubic Connection**: 10+ structural parallels, p < 10^-6 for independent convergence
**See**: [Numogram Connection](/docs/03-results/29-numogram-architecture)

### Pandemonium System

CCRU concept: distributed architecture where autonomous entities interact through feedback to create emergent order from chaos.

**Qubic Implementation**: Aigarth's evolutionary neural network
**Definition**: "Both the place of all demons and the system of all feedbacks"

### Syzygies

In Numogram: paired zones summing to 9 (0+9, 1+8, 2+7, 3+6, 4+5)

**Qubic Parallel**: row%8 complementary classes in Aigarth neural tissue

### Theory-Fiction

CCRU genre blending theoretical philosophy with fictional narratives, treating ideas as operational systems.

**Example**: Numogram as blueprint for AGI (theory becomes working code)

| Term | Definition |
|------|------------|
| **Ternary** | Base-3 computing (-1, 0, +1) |

| **Epoch** | Collection of ticks |
| **Hash160** | SHA256 + RIPEMD160 |
| **P2PKH** | Pay-to-Public-Key-Hash (legacy address) |

### Mirror Wallet Operations

| Term | Definition |
|------|------------|
| **Mirror Palindrome** | Hash160 transformation: first 10 bytes + reversed first 10 bytes. Preserves address prefix! |
| **Reverse Hash160** | Byte-order reversal of Hash160 value |
| **Bitwise Complement** | Each byte becomes (255 - byte) |
| **Rotate +N** | Add constant N to each byte (mod 256) |
| **XOR Operation** | Bitwise XOR between two Hash160 values |
| **Swap Halves** | Exchange first and last 10 bytes |
| **ECC Public Key Addition** | Elliptic curve point addition on public keys (Craig Wright trick) |
| **Vanity Address** | Bitcoin address with custom prefix (e.g., "1CFB") |

**Important**: Mirror operations create valid addresses but we have no private keys for them.
