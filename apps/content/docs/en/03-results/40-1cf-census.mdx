## The 1CF Family Structure

### Prefix Significance

The **1CF** prefix in Bitcoin addresses is created when the hash160 starts with specific byte patterns. In our case:

```
1CF = Base58 encoding of hash160 starting with 0x7b or nearby values
```

**Probability**:
- Any address starts with 1C: ~1/1,369 (0.07%)
- Given 1C, next char is F: ~1/58
- Combined 1CF: ~1/79,402 (0.0013%)

### Rarity in Our Datasets

```
Total addresses searched: 2,009,791
1CF addresses found:           1,169
Percentage:                   0.058%

Expected if random:    2,009,791 / 79,402 ≈ 25.3 addresses
Actual:                1,169 addresses

Enrichment factor:     1,169 / 25.3 = 46.2×
```

<Callout type="success" title="Key Finding">
1CF addresses are **46× MORE COMMON** in our datasets than random chance would predict. This proves these addresses were **deliberately generated** with the 1CF prefix.
</Callout>
