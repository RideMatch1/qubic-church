## Code Style Indicators (Tier 3)

### Observed Patterns

Analysis of early Bitcoin source code reveals stylistic markers:

**1. Quadruple-Slash Comments**:

```cpp
//// issue here: it doesn't know the version.
//// is this all we want to do...
//// or maybe
```

**2. Plural Pronoun Usage**:

```cpp
// We need to...
// We should...
```

**3. Integer-First Philosophy**:

Preference for integer arithmetic over floating-point, consistent with CFB's documented coding practices.
