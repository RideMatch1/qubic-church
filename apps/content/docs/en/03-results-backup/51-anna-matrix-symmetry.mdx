## The Mathematical Discovery

### The Symmetry Formula

For 99.6% of all positions in the Anna Matrix:

```python
matrix[r][c] + matrix[127-r][127-c] = -1
```

This is equivalent to the **Two's Complement** identity:

```python
value + (~value) = -1
value + (value XOR 0xFF) = -1
```

## Verification Method

### Script: `research_matrix_negation.py`

```python
import json
import numpy as np

# Load Anna Matrix
with open('apps/web/public/data/anna-matrix.json') as f:
    data = json.load(f)
    matrix = np.array(data['matrix'])

# Count symmetric pairs
symmetric_count = 0
anomaly_count = 0

for r in range(64):  # Only need to check half
    for c in range(128):
        val = matrix[r][c]
        mirror_val = matrix[127-r][127-c]

        if val + mirror_val == -1:
            symmetric_count += 1
        else:
            anomaly_count += 1

# Each pair covers 2 cells
print(f"Symmetric cells: {symmetric_count * 2}")
print(f"Anomaly cells: {anomaly_count * 2}")
print(f"Symmetry rate: {symmetric_count * 2 / 16384 * 100:.2f}%")
```

### Output

```
Symmetric cells: 16316
Anomaly cells: 68
Symmetry rate: 99.59%
```
