   > "If Satoshi was CPU mining at ~7 MH/s, finding a hash with 43 zeros would take ~4 months. With extraNonce=4, it should take seconds."

3. **Possible Explanations**:
   - Satoshi **pre-computed** the Genesis hash with specific properties
   - The timestamp was **backdated** to embed a message (The Times headline)
   - The Genesis **address might itself be a nonce** - chosen for mathematical properties

4. **Statistical Impossibility**:
   ```
   Expected extraNonce for 43 zeros: ~2048
   Actual extraNonce: 4
   Probability of this by chance: < 0.2%
   ```

### What is E8 Theory?

**E8** (or E₈) is the largest exceptional simple Lie group, a 248-dimensional mathematical structure. In 2007, physicist **Garrett Lisi** proposed "An Exceptionally Simple Theory of Everything" using E8 to unify all fundamental forces.

Key properties:
- 248 dimensions
- 8-dimensional root lattice
- Contains patterns for all known elementary particles
- Potentially encodes quantum gravity
