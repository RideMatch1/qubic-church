## Die Bedeutung von -1

```
matrix[r, c] + matrix[127-r, 127-c] = -1

**Die Zahl -1 ist die Symmetrie-Konstante!**

Wenn eine Position UNKNOWN ist, dann enthält ihr Spiegel die Konstante -1:

```
