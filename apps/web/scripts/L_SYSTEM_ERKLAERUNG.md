# DAS L-SYSTEM: Ein öffentliches Blockchain-Protokoll

---

## Was ist das L-System?

Das L-System ist ein **informelles Messaging-Protokoll** in der Bitcoin Blockchain, bei dem Nachrichten im Format `L[nummer],[nachricht]` in OP_RETURN-Transaktionen geschrieben werden.

### Technisch:
- Jeder kann eine L-Nachricht posten
- Man braucht nur eine Bitcoin-Transaktion mit OP_RETURN
- Die L-Nummer ist frei wählbar (kann überschrieben werden)
- Das @-Symbol referenziert andere L-Nummern oder Indizes
